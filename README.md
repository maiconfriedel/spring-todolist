# Spring Todo List

API desenvolvida utilizando Java 17 com Spring Boot 3, com CRUD (Create, Read, Update, Delete) de uma lista de tarefas.

## Tecnologias Utilizadas

- Spring Web
- Spring Data JPA
- Spring Test
- MySQL
- H2 Database (Testes)
- Spring Validation
- Swagger UI Docs
